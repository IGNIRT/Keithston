# Keithston cайт пекарни с адаптивным дизайном и проработанной структурой.
Тот самый хруст на весь район. Печем честно, кормим вкусно. Хлеб, кофе и ничего лишнего.
# https://www.figma.com/design/KrNEa5bMBPMVuINwm5zhpX/Bakery-Website-Ui--Community-?node-id=0-1&p=f&t=ImS64sTgVuXaJZwL-0

Этот проект — результат работы над макетом из Figma. Я старался сделать интерфейс чистым, удобным для пользователя и адаптивным под разные экраны. Это была моя первая практика в работе с готовым дизайном и превращении его в живой код.

## Задачи 
- Вёрстка всех блоков по макету
- Создание адаптивной сетки для мобильных и десктопных версий
- Настройка навигации и интерактивных элементов

## Технологии
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript
